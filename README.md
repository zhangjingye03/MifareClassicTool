执信中学饭卡读取程序
=========================

这是一个能在带有NFC硬件的安卓设备上读取执信中学饭卡部分信息的APP。

* **下载地址**
  (https://github.com/zhangjingye03/MifareClassicTool/releases)



功能
--------

* 读取执信中学饭卡的0~2号扇区并显示
* 识别执信中学饭卡的部分信息（卡号、余额、上次消费、消费次数、连续消费次数、上次消费月份等）
* 检测执信中学饭卡是否损毁
* 检测是否为执信中学饭卡
